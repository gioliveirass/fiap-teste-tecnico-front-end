# Sobre o projeto

O presente projeto consiste em uma aplicação WEB responsiva desenvolvida durante um teste técnico da empresa FIAP.

# Tecnologias utilizadas

- Next.js
- TypeScript
- Framer Motion
- Sass

# Executando o projeto localmente

Após clonar o repositório, abra a pasta raiz e execute os seguintes comandos:

```bash

# Para baixar as dependências
yarn

# Para inicializar o ambiente de desenvolvimento
yarn dev
```
